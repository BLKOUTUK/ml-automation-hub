# ML Automation Hub

Infrastructure for machine learning experimentation and rapid MVP development.

## Components

- **n8n workflows**: Automated processes for data collection and processing
- **Connectors**: APIs for MCP servers via Comprosio
- **Models**: Saved ML models and evaluation scripts
- **Notebooks**: Jupyter notebooks for experimentation

## Setup

1. Clone this repository
2. Configure environment variables in `config/settings.json`
3. Follow setup instructions in the docs folder
